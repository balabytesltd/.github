# .github

This repository contains default community health files and organization-wide configurations for [Balabytes Ltd](https://github.com/balabytesltd).

## Overview

The `.github` repository is a special GitHub repository used to share default files and settings across all repositories in the organization. Files placed here will be used as defaults for any repository that does not have its own version of these files.

## Contents

This repository may include:

- **Issue templates** – Default templates for bug reports, feature requests, and other issues.
- **Pull request templates** – Default templates to guide contributors when submitting pull requests.
- **Contributing guidelines** (`CONTRIBUTING.md`) – Instructions for how to contribute to projects in this organization.
- **Code of conduct** (`CODE_OF_CONDUCT.md`) – Community standards and expectations.
- **Security policy** (`SECURITY.md`) – How to report security vulnerabilities.
- **Support resources** (`SUPPORT.md`) – Where to get help with projects.
- **Funding information** (`FUNDING.yml`) – Links to sponsorship or funding platforms.

## Usage

Any file placed in this repository at the supported path will automatically apply to all repositories in the **Balabytes Ltd** organization that do not define their own version of that file.

For more information, see the [GitHub documentation on default community health files](https://docs.github.com/en/communities/setting-up-your-project-for-healthy-contributions/creating-a-default-community-health-file).
